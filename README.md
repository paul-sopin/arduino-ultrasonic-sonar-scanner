# arduino-ultrasonic-sonar-scanner
Simple arduino sonar scanner that detects objects using an ultrasonic sensor attached to a servo.

What it is:
- Rotating ultrasonic sonar scanner on a micro servo powered by an Arduino Uno
- The scanner rotates 180 degrees, and whenever an object is within a specific radius it will react:


- over or equal to 20cm: No buzzer sounds, green LED
- between 10cm and 20cm: No buzzer sounds, blue LED
- under 10cm: The buzzer sounds, and the LED goes red
